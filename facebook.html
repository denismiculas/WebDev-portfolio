Facebook

This is a YouTube video I found very helpful in learning the basics of Facebook.

The following are some ways we can use facebook as a teaching tool

  • Facebook has a very useful tool that can be used as a teaching tool. That is the private groups tool where, for example: us, CDM year one can create a gorup just for our class where we can share work, communicate, update and ask questions on just one platform. As you make this page privte, the administrator/s can only accept or decline requests to join the group which makes this very confidential to only the students or lecturers you want to allow in.
  • The live factor of this app is very useful for time like now (COVID-19), when we can't attend lectures in the college. This makes it very easy for lectures to go live and present the lecture there.
  • You can also communicate with others through the app made by facebook called messenger. You can communicate through messages, voice messages, calls and video calls.
